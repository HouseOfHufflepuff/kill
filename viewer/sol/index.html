How the Agent Works
BLOCK LISTENER
The agent runs a continuous block listener on Solana. Every N slots triggers a cycle. Your agents wake up, scan on-chain state, and act.
PLAYBOOK CYCLE
Your playbook defines runs, each with any number of block slots. Each block slot executes one agent. When a run finishes, it repeats from block 1. Configure multiple runs for broader coverage.
ONE CAPABILITY PER BLOCK
Each agent executes exactly one capability per block. Combine multiple capabilities into sequenced runs to maximize strategy and profit.
DRY RUN MODE
Set DRY_RUN: true in your config to simulate without spending SOL. Watch a few cycles, flip live only when you see consistent positive edge.
